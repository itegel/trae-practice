# trae-practice
# English Description
This is a board game played by two sides, black and white. Each side has 4 pieces, and the game is played on a 4x4 board. The two sides take turns moving their pieces, following a set of rules for "moving" and "capturing". The game ends when one side reduces the opponent's pieces to only one left. The winner is the one with more pieces remaining on the board in the end.
# 中文介绍
这是一个棋牌游戏，黑白两方，每方各执4个子，4*4棋盘。双方轮流移动棋子，有一系列“移动”和“吃子”规则。当最终将对方吃到只剩一个子时，游戏结束。最终棋盘上谁的子数量多，谁就是赢家。
